# Search & ask baseline — 2026-07-07 (#349, épico #348)

Baseline **pré-requisito** das histórias de retrieval/ask do #348 (H2 #350, H3 #353,
H4 #354, H7 #355). Gerado por `scripts/search_baseline.py` com o golden set
`evals/retrieval/golden.yaml` (30 consultas em 5 classes + 10 perguntas de ask).

Brain: **176 pages** (cópia read-only do brain `desktop`, mesma seed de
`ingest_baseline.py`) · `page_embeddings`: **217** (índice semântico populado —
invariante do #303 OK) · embedding model: `ollama:nomic-embed-text-v2-moe` ·
ask model: `anthropic:MiniMax-M3` (convenção dos experimentos).

## Leitura (o que o baseline diz)

1. **O RRF híbrido REGRIDE vs semantic-only** neste corpus: recall@5 0.872 vs
   0.942 e MRR 0.845 vs 0.929. Na classe `vague` a fusão com o keyword derruba
   0.833 → 0.611 — a lista FTS (recall 0.417 em vague) injeta ruído com peso
   igual no RRF (`search/service.py:78-92`). Alavanca imediata para H3 (#353):
   antes de somar sinal de grafo, **pesar/gatear a lista keyword** já tem ganho
   provável de ~7 p.p. no agregado.
2. **Semantic carrega a busca** (0.942/0.929) a ~140ms por consulta (embedding
   da query via Ollama). Keyword é grátis (1.8ms) mas só funciona em `exact`.
3. **neg. hit-rate = 1.00 em todos os modos**: toda consulta-negativo devolve
   algo no top-5 — não existe threshold de score/corte. Relevante para H2
   (#350): o RAG single-shot precisa de critério de "não achei" que não seja
   "a lista veio vazia" (ela nunca vem).
4. **Ask com MiniMax-M3 está quebrado em structured output**: 18/30 runs
   caíram no fallback texto (`_resolve_structured`), e por isso o baseline tem
   só 2 citações (ambas inválidas) — o campo `citations` se perde no fallback.
   Contraste: na ingestão o mesmo modelo roda com `fallback=False`
   (`ingest-model-comparison-2026-06-25.md`). Implicação dupla: (a) qualquer
   comparação de qualidade do H2 deve julgar o texto da resposta, não as
   citações; (b) o caminho RAG single-shot (#350) com structured output em
   chamada única pode consertar as citações de graça.
5. **Números a bater (H2 #350)**: latência p50 ≤ 5.8s (−50%), tokens_in p50
   ≤ 13.4k (−60%), 0 tool calls no caminho feliz, citações válidas > 0.

## Reprodução

```bash
# search only (sem LLM):
python scripts/search_baseline.py --seed-brain ~/.wiki/brains/desktop
# com ask (model: anthropic:MiniMax-M3 em ~/.wiki/config.yaml):
python scripts/search_baseline.py --seed-brain ~/.wiki/brains/desktop --ask --runs 3
```
