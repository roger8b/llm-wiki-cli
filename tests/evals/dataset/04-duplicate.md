# Embedding Vectors

An embedding vector is a dense, fixed-length list of floating point numbers that
represents the meaning of a piece of data such as a word, sentence, or image.
Items with similar meaning are mapped to vectors that sit close together in a
high-dimensional space, so geometric distance approximates semantic similarity.

This is exactly the same idea as a vector embedding: a model encodes the input
into a numeric vector, and nearest-neighbour search over those vectors powers
semantic search and retrieval. Because this concept already exists in the wiki
under another name, the right move is to enrich the existing page rather than
create a second one.
